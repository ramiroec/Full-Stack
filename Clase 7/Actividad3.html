Manipulación de Cadenas en JavaScript

trim()

Elimina los espacios en blanco al inicio y al final de un string.

let text = " Hello World! ";
let trimmedText = text.trim();
console.log(trimmedText); // "Hello World!"

concat()

Concatena dos o más strings.

let text1 = "Hello";
let text2 = "World";
let result = text1.concat(" ", text2);
console.log(result); // "Hello World"

charAt()

Retorna el carácter en una posición específica.

let text = "Hello";
let char = text.charAt(1);
console.log(char); // "e"

repeat()

Repite un string un número específico de veces.

let text = "Hello";
let repeatedText = text.repeat(3);
console.log(repeatedText); // "HelloHelloHello"
